# Chapter 31 - Talking to the Web

Example code for Chapter 31 of *Introduction to the Aussom Programming Language*.

**Needs internet access** - the example calls a live test API (jsonplaceholder.typicode.com).

## Files

- **`web.auss`** - GET and POST over HTTP with the `http` module, sending/receiving JSON, and handling error statuses and connection failures.

## Running

```
aussom web.auss
```

# Chapter 25 - Files and the Filesystem

Example code for Chapter 25 of *Introduction to the Aussom Programming Language*.

## Files

- **`files.auss`** - reading/writing text (`read`/`write`/`readLines`), path parts, and directories.
- **`buffer.auss`** - raw bytes with `Buffer` and `file.writeBinary`/`readBinary`.

## Running

```
aussom files.auss
aussom buffer.auss
```

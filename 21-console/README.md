# Chapter 21 - The Console and Core Helpers

Example code for Chapter 21 of *Introduction to the Aussom Programming Language*.

## Files

- **`corehelpers.auss`** - console output levels, formatted output (`os.sprintf`/`printf`), `lang.type`, and `json`.
- **`ask.auss`** - reading input from the console with `os.readLine`.
- **`saveobject.auss`** - marshalling an object to disk with `pack` and rebuilding it with `json.unpack`.

## Running

```
aussom corehelpers.auss
aussom ask.auss
aussom saveobject.auss
```

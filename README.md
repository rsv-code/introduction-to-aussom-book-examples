# Introduction to the Aussom Programming Language — Example Code

This repository holds the complete, runnable example code for the book
**_Introduction to the Aussom Programming Language: A Hands-On Guide for New
Programmers_** by Austin Lehman.

Each chapter's examples live in their own numbered directory (for example,
`04-your-first-program/`), matching the chapters in the book. Most directories
contain one or more Aussom programs (`.auss` scripts or `.aus` files) and a short
`README.md` describing them.

## Getting the book

You can get the book and learn more about the Aussom language at
**[aussom-lang.com/books](https://aussom-lang.com/books)**.

## Running an example

Install the Aussom CLI (see [aussom-lang.com/download](https://aussom-lang.com/download)) so the
`aussom` command is on your `PATH`, then run any example from its directory:

```bash
cd 04-your-first-program
aussom hello.auss
```

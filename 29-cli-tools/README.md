# Chapter 29 - Building Command-Line Tools

Example code for Chapter 29 of *Introduction to the Aussom Programming Language*.

## Files

- **`greet.auss`** - a small CLI tool built with `cliparser`: value options, a flag, auto-generated help, and validation.

## Running

```
aussom greet.auss -n Ada
aussom greet.auss --name Grace --count 3 --upper
aussom greet.auss --help
```

# Chapter 32 - Working with Databases

Example code for Chapter 32 of *Introduction to the Aussom Programming Language*.

Uses the **SQLite** driver bundled with Aussom, so no database server is needed.

## Files

- **`contacts.auss`** - connect with `jdbc`, create a table, insert/query/update rows (creates a local `contacts.db` file).

## Running

```
aussom contacts.auss
```

# Chapter 17 - Inheritance and Polymorphism

Example code for Chapter 17 of *Introduction to the Aussom Programming Language*.

## Files

- **`animals.auss`** - an `Animal` base class with `Dog` and `Cat` subclasses that
  override `speak()`, plus polymorphism via a shared `describe()` method.

## Running

```
aussom animals.auss
```

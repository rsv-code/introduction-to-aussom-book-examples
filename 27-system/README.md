# Chapter 27 - System and OS Access

Example code for Chapter 27 of *Introduction to the Aussom Programming Language*.

## Files

- **`system.auss`** - system info (`sys`), environment variables, and running a command (`os`).
- **`greet.aus`** - command-line arguments (`main(args)`) and exit codes (`os.exit`).

## Running

```
aussom system.auss
aussom greet.aus World
aussom greet.aus
```
